in order to run the aiproject.ipynb file;
you need to open anaconda prompt and write the following lines in order to download the libraries used in the project file
1.conda install -c conda-forge textblob
2.python -m textblob.download_corpora
3. conda install -c conda-forge spacy
4.python -m spacy download en
5. conda install -c conda-forge keras
6.pip install matplotlib
7.pip install seaborn

after downloading these libraries, open the file in jupyter notebook,
now run the file.
